# 03-7 Python的內建函式-open


## 1. 內建函式open()

#### open()可以開啟檔案. 



#### 程式範例
```javascript
# 開啟檔案(r:只讀)
file = open('data.txt','r')  

# 每行依次讀入
for line in file.readlines():  
    line = line.strip()  #去除頭尾空白及\n 
    print(line)

print('-'*30)

# 關閉檔案
file.close()                  
```
