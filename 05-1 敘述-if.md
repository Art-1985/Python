# 05-1 敘述-if


## 1. if

#### 用if進行不同情況的處理.



#### (1)程式範例
```javascript
score=int(input('成績:'))

# if判斷
if score>=90:
    rank='優等'
elif score>=80 and score<90:
    rank='甲等'
elif score>=70 and score<80:
    rank='乙等'
elif score>=60 and score<70:
    rank='丙等'
else:
    rank='丁等'    

print('等第={}'.format(rank))          
```


執行結果:
```
成績: 90
等第=優等
```
